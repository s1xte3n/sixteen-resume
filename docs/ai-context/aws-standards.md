# AWS Standards

## Lambda

- Use environment variables
- Use structured logs
- Use retries where appropriate
- Configure memory intentionally
- Configure timeouts intentionally

## DynamoDB

- Enable PITR
- Use partition key strategy carefully
- Avoid scans
- Use GSIs intentionally

## API Gateway

- Enable CORS
- Validate requests
- Use proper status codes

## IAM

- Least privilege only
- Avoid wildcard permissions
- Separate roles by responsibility
